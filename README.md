# one4all script

Linux install/config script all for one

一个目标将Linux的安装/配置操作集成到一个脚本工具中的项目。

Linux发行版很多，操作各有不同，但都有固定的规则，将这些规则固化到脚本中，让一个脚本解决这些安装/配置问题吧。

任何人都可以使用Linux 娱乐/软件开发。
